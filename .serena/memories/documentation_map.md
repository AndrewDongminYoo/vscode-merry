# Documentation map for vscode-merry

- `CLAUDE.md`: repo-wide architecture, parser semantics, commands, and product constraints.
- `AGENTS.md`: root navigation map and cross-cutting editing guidance.
- `src/AGENTS.md`: runtime module boundaries and edit hints.
- `src/test/AGENTS.md`: test suite responsibilities and guardrails.
- `test-workspace/AGENTS.md`: committed fixture contract.
- Serena memories: short persistent reminders for commands, style, completion checks, and this document map.

## Usage rule

- Read `CLAUDE.md` when you need the why.
- Read the nearest `AGENTS.md` when you need the local boundary and file map.
- Read Serena memories when you need compact reminders without reloading the repo docs.
